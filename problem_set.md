# 经典面试题

每种类型的题目保留典型三道

## 优先队列

### 前 K 个高频元素

给你一个整数数组 nums 和一个整数 k ，请你返回其中出现频率前 k 高的元素。你可以按 任意顺序 返回答案。leetcode 例题：<https://leetcode.cn/problems/top-k-frequent-elements/description/>

## 动态规划

### 最大子数组和

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。子数组是数组中的一个连续部分。leetcode 例题：<https://leetcode.cn/problems/maximum-subarray/description/>
